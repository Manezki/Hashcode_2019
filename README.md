# Google hashcode submission

Submission for [Google Hashcode](https://codingcompetitions.withgoogle.com/hashcode) challenge 2019.

While we only reached ranking 627, we were happy to finish as the **#1 in Finland**.

----

## Solution

Makes heavy use of set-operations, and applying greedy algorithms with stopping rules.
